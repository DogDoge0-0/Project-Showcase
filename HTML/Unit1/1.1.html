1.1: Make a Recipe


This page was the first one I ever made. We learned about the img (image), ul (unordered list), ol (ordered list), li (list item element), and header tags (h1, h2, etc.). As you can tell, some stuff has been remastered since then. The page is in dark mode, the image has a border, and there are some hr (horizontal rule) tags splitting up the various parts of the page. Sadly, I did not have the foresight at the time to preserve the page in its original, un-remastered form, and as such, it has been lost to time. The original version of this project was last edited on September 28, 2022.
